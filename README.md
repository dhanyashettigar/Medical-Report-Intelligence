# Medical-Report-Intelligence
An AI-powered assistant that reads your medical report (PDF) and explains it in simple language. It highlights abnormal values, gives a health summary, and suggests lifestyle tips — like having a doctor translate your report into plain English.
Built with LangChain, FAISS, HuggingFace embeddings, and Groq's LLaMA 3.3 model, with a Streamlit web app for easy use.
